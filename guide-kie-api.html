Kie.ai API 키 설정 가이드 🤖

이 가이드는 kie.ai에서 API 키를 발급받아 프로그램에 설정하는 과정을 안내합니다.

1단계: kie.ai 웹사이트 접속 및 로그인

먼저, kie.ai 웹사이트에 접속하여 회원가입 또는 로그인을 진행합니다.

kie.ai 메인 페이지 캡쳐

2단계: API 키 관리 페이지로 이동

로그인 후, 우측 상단의 프로필 아이콘을 클릭하고 메뉴에서 'API Key' 또는 'API 관리'를 선택합니다.

프로필 메뉴에서 API Key를 선택하는 캡쳐

3단계: 새 API 키 생성

API 키 관리 페이지에서 'Create New Key' 또는 '새 키 생성' 버튼을 클릭합니다.

키의 용도를 알아보기 쉽게 이름(Label)을 입력하고 'Create' 버튼을 누릅니다. (예: MusicVideoCreator)

새 키를 생성하는 화면 캡쳐

4단계: 생성된 API 키 복사

새로운 API 키가 생성되면 화면에 표시됩니다. 이 키는 다시 볼 수 없으므로 반드시 안전한 곳에 보관하세요.

키 옆에 있는 복사(Copy) 아이콘을 클릭하여 키를 클립보드에 복사합니다.

생성된 키와 복사 버튼 캡쳐

5단계: 프로그램에 API 키 붙여넣기

YouTube Content Creator 프로그램을 실행하고 '⚙️ 환경설정' 탭으로 이동합니다.

'Kie.ai API Key' 입력란에 방금 복사한 API 키를 붙여넣습니다.

'테스트' 버튼을 눌러 "Kie.ai API 연결에 성공했습니다." 메시지와 함께 남은 크레딧이 표시되는지 확인합니다.

마지막으로 '모든 설정 저장' 버튼을 눌러 키를 프로그램에 완전히 저장합니다.

프로그램 환경설정 탭에 키를 붙여넣고 테스트하는 캡쳐
