Gemini API 키 발급 완벽 가이드 🔑

Gemini API 키, 왜 필요할까요?

최신 AI 기술을 활용한 콘텐츠 생성을 위해서는 강력한 언어 모델이 필수적입니다. Google의 Gemini 모델, 특히 Gemini 1.5 Flash는 뛰어난 성능과 합리적인 비용으로 다양한 애플리케이션에 활용될 수 있습니다. 저희 YouTube 콘텐츠 생성 프로그램에서도 가사 생성, 프롬프트 작성, 제목 및 설명 생성 등 핵심 기능에 Gemini API를 사용하고 있습니다.

이 가이드에서는 Google AI Studio를 통해 Gemini API 키를 발급받는 과정을 단계별로 상세히 안내해 드립니다. 차근차근 따라오시면 누구나 쉽게 API 키를 얻으실 수 있습니다!

준비물

  • Google 계정 (Gmail 계정 등)
  • 인터넷 연결이 가능한 PC 또는 모바일 기기

Gemini API 키 발급 단계

1단계: Google AI Studio 접속

먼저, 웹 브라우저를 열고 Google AI Studio 페이지로 이동합니다. 아래 버튼을 클릭하시거나, 주소창에 직접 입력해주세요.

Google AI Studio 바로가기

또는 주소창에 https://aistudio.google.com/를 입력합니다.

Google AI Studio 초기 화면입니다. (이미지는 예시이며, 실제 화면과 다를 수 있습니다.)

2단계: Google 계정으로 로그인

Google AI Studio에 접속하면 Google 계정으로 로그인하라는 메시지가 나타날 수 있습니다. 기존에 사용하시던 Google 계정으로 로그인해주세요. 만약 계정이 없다면 새로 생성해야 합니다. 로그인 후 서비스 약관 동의 등의 절차가 있을 수 있으니, 안내에 따라 진행해주세요.

Google 계정 로그인 화면입니다. (이미지는 예시이며, 실제 화면과 다를 수 있습니다.)

3단계: API 키 생성 또는 확인

로그인이 완료되면 Google AI Studio의 메인 대시보드로 이동합니다. 여기서 API 키를 생성하거나 기존에 생성된 키를 확인할 수 있습니다. 화면 좌측 메뉴에서 "Get API key" 또는 "API 키 받기"와 유사한 메뉴를 찾아 클릭합니다.

대시보드에서 API 키 관련 메뉴를 찾는 예시입니다. (이미지는 예시이며, 실제 화면과 다를 수 있습니다.)

새 API 키 생성

만약 기존에 생성한 API 키가 없다면, "Create API key in new project" 또는 "새 프로젝트에서 API 키 만들기" 버튼을 클릭하여 새로운 키를 생성할 수 있습니다.

기존 API 키 확인

이미 API 키를 생성한 적이 있다면, 해당 프로젝트 목록과 함께 API 키가 표시됩니다. 필요한 API 키를 여기서 바로 복사할 수 있습니다.

4단계: API 키 복사

API 키가 생성되거나 화면에 표시되면, 해당 키 문자열을 복사합니다. 보통 키 옆에 복사 아이콘(📋)이 제공되어 편리하게 복사할 수 있습니다.

생성된 API 키와 복사 버튼 예시입니다. (이미지는 예시이며, 실제 화면과 다를 수 있습니다.)

주의: API 키는 비밀번호와 같이 중요한 정보입니다. 타인에게 노출되지 않도록 안전하게 보관해주세요.

API 키, 프로그램에 어떻게 적용하나요?

발급받은 Gemini API 키는 저희 YouTube 콘텐츠 생성 프로그램의 [환경설정] 탭에서 [Gemini API Key] 입력란에 붙여넣고 저장하시면 됩니다. 저장 후에는 프로그램의 모든 Gemini 기반 AI 기능을 정상적으로 이용하실 수 있습니다.

프로그램 내 API 키 입력 화면 예시입니다. 입력 후 정상 작동하는지 테스트 버튼을 눌러 확인합니다.
