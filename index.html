1

콘텐츠 주제 설정

AI가 생성할 콘텐츠의 주제와 방향을 명확히 설정합니다.

2

자동 생성 실행

3

결과 검토 및 수정

생성된 콘텐츠를 검토하고 필요시 수정합니다.

4

플랫폼 업로드

완성된 콘텐츠를 해당 플랫폼에 자동으로 업로드합니다.

API 설정 가이드

Gemini API 키 발급 가이드

콘텐츠 생성을 위한 Google Gemini API 키를 발급받는 방법을 안내합니다.

YouTube API 설정 가이드

자동 업로드를 위한 YouTube Data API v3 키 및 OAuth 2.0 설정을 안내합니다.

Kie.ai API 설정 가이드

PiAPI를 대체하는 Kie.ai의 이미지/비디오 생성 API 키 설정을 안내합니다.

자주 묻는 질문

네, 대부분의 프로그램은 AI 모델(예: Gemini, Suno)을 활용하기 위해 개인 API 키가 필요합니다. 프로그램 내 설정 가이드를 참고하여 발급 및 설정할 수 있습니다.

현재 프로그램은 Windows 10/11 환경에 최적화되어 있습니다. macOS 버전은 추후 지원될 예정입니다.

AI를 통해 생성된 콘텐츠의 저작권은 기본적으로 사용자에게 귀속됩니다. 상업적 이용이 가능하지만, 사용된 AI 서비스의 약관을 최종적으로 확인하는 것을 권장합니다.

네이버 카페 가입 시, 질문 답변란에 원하시는 '인증 ID'를 남겨주세요. 제가 ID를 확인하고 등업해드리면 질문 게시판을 이용하실 수 있습니다. 쉽고 무료로 더 체계적인 지원을 받아보세요!
